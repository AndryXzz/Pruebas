El elemento <div> permite agrupar un conjunto de elementos juntos en un cuadro de nivel de bloque. por ejemplo, puede crear un elemento <div> para que contenga todos los elementos del encabezado del sitio (el logotipo y la navegación), o puede crear un elemennto <div> para que contenga información en bloques

En un navegador, El contenido del elemento <div> comenzará en una nueva línea, pero aparte de esto, no hará ninguna diferencia en la presentación de la página. sin embargo, el uso de un atributo ID o class en el elemento <div> significa que puede crear reglas de estilo CSS para indicar cuánto espacio debe ocipar el elemento <div> en la pantalla y cambiar la apariencia de todos los elementos contendios en él. También puede facilitar en seguimioento del código si ha utilizado elementos <div> para contener cada sección de la página.

Dado que puede haber varios otros elementos dentro de un elemento <div>,puede ser útil agregar un comentario después de la etiqueta </div> de cierre. Esto le permite ver claramente a qué etiqueta de apertura se supone que corresponde.
